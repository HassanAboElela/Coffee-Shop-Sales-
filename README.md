# **Coffee Shop Sales**

### **Objective**
Developed an interactive Excel dashboard to analyze coffee shop sales performance, monitor revenue trends, and identify high-growth product categories.

### **Technical Implementation**
* **Data Cleaning:** Handled missing values and ensured data consistency for accurate reporting.
* **Data Modeling:** Utilized Power Pivot to create relationships and advanced calculations.
* **Interactive Visualization:** Built a dynamic dashboard with slicers for real-time monthly and location-based filtering.

### **Key Insights**
* **Product Performance:** Coffee and Tea are the primary revenue engines, with Barista Espresso as the top-selling product.
* **Growth Trends:** Identified a consistent upward trend in revenue, reaching a peak in June.
* **Regional Analysis:** Sales are evenly distributed across Astoria, Hell's Kitchen, and Lower Manhattan.
